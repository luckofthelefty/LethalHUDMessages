# Changelog

## 1.0.0

- Initial release
- Two display styles: Tip (bottom-right popup) and GlobalNotification (bottom-center banner)
- Color-coded GlobalNotification backgrounds (red for deaths, yellow for events)
- Message queue with 4-second minimum display time
- Death messages for all 16 causes of death
- Monster-specific kill messages for 21+ monsters
- Expanded monster encounter messages (25+ encounter types)
- Event messages (critical damage, ship leaving, vote to leave, teleporter, quota, turret)
- Modded/custom enemy support
- Debug test mode (F8-F11 hotkeys)
